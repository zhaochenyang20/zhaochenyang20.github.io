10 月 1 日

晚上成外聚餐，甚是快乐。

从我大一入学，到去年，到如今，竟然有三年之久了。

有人永远年轻，而我的本科生涯却已经走到尽头。

饭桌上听低年级小朋友们谈论内卷，谈论绩点，谈论保研。我谈起这几年的感受，讲出了我的顾虑与反叛。这让我想起了高中时约翰对我说的话，“你天生有反骨”。

现在想来，成外在我心中的形象就犹如中国。每年都喊着赶英超美，这么多年也没见到真的成功了，只是拖死了苏联。每年都有人喊着中国要完，从九十年代到如今，年年都是国将不国，然而不也缝缝补补继续向前了。

成外也是如此吧，也许我没有见过所谓全盛时的成外，然而在我心中，那些人，那些事情，那些憧憬，那些心心念念，那些东五楼的日夜，那些雨后淋湿的校服，那些熬穿的深夜，那些没能实现的遗憾，会永远在记忆里的角落，无时无刻不在塑造着，在影响中，在热烈着。

我越发感到时间的流逝，本科的生涯所剩无几，而我即将远航。

希望能用这些文字纪念下如今的自己。

10 月 2 日

晚上 22:40

这一两天在 ANN 上找回了一直以来的快乐感，由衷发现我的内心喜爱的方向果真是机器学习。今天和阿飞讨论参数设置，讨论学习率太高导致的动荡和太低根本学不动；还有类似 weight_decay 和 momentum 这两个参数我完全不能理解，或许明天需要去公司找詹老师他们好好请教。我深知自己水平拉胯完全没法匹配公司给我的任务，而有时还碍于虚荣难以求助。

可是，一直这样，就和以前一样了；不能停在过去啊。

调参过程中，我发现 copilot 补全的代码居然会不对齐，发了条很乐的朋友圈，Weikai Yang（刘世霞老师的博士生，答疑坊大佬） 评论到需要掌握 product 和 subprocess，我知道前者，但是从未用过，而后者我下午刚刚听博文讲过，却没有来得及学习。

那么，下次调参时，我该学会更多东西，这次学习了不少对于参数的理解（假设我能明天找到人讨论这些参数的意义以及学会每一层求导在做什么且向前传了什么，而不是故步自封碍于虚荣），在技术上实际上没有太多进步。

总之，是个好的开始，先把代码写好，然后再一天收获。

10 月 3 日

次日 0:09

如果用什么来表述昨天的生活，我想，是充实和幸福感。

这种感觉来自于新知与前进。

上午推进 ANN 的 report 和公司的项目，发现经过前几周的反复挣扎加上昨日的修整，今天的借口很快写好了，而且写的鲁棒。此外，我还成功第一次在公司的主机上提交了 srun bash 脚本；之前对于 srun 的抗拒，可能是恐惧于没法及时看到 tqdm 的提示信息，然而这次我发现完全可以见到，而且节点的运行速度显著快于管理机；果然以后的任务是小规模实验，大规模交任务。

此后，开始写报告，从 wandb 上截图，而后用 overleaf 编译，终于实现了我学习 latex 的梦想。我从大一寒假试图开始学习 latex，可惜我那时连 markdown 都不会，图床尚且不理解，何谈 latex 呢？我不由得感慨我学习能力的低下，这点和鲁大师对比鲜明。他入学前的暑假学完了 C++ Primer Plus 和 Python 基础语法，说不定我大学后第一次见到他时，他的 Python 水平甚至强于还没在 DISCOVER 当牛马的我；一年下来，我的 Python 的确进步显著，或许得感谢 DISCOVER 让我当了一年牛马；当然，更有可能是我的学习效率只能当牛马。

总之，今天第一次用 Latex 成功写出了像样的实验报告，并且写的很快乐。在写的的过程中，我参考了 Nauy 的报告，让我大受震撼。Nauy 的报告语言非常口语化，然而言之有物，一针见血指出了某些关键参数和模型的疏漏，而我得翻阅大量资料且回顾在人智导上打下的非常薄弱的基础才行。他的所见所闻让我佩服，不愧是能够在本科这个阶段就做出如此学术成功的人。然而，人比人注定气死人，何况这是在贵系。Nauy 的能力已经从他集训队出身可见一斑，而在高中同期，刘好已经开始搞学术了，我还在玩泥巴。

与之相比，子睿的报告和我非常像，看得出并没有良好的深度学习功底，然而可见态度之端正和功夫之细致。如此想来，我和子睿是相似的，是靠谱的队友，也能成为团队的领袖，然而没法成为团队的技术灵魂；这样的技术灵魂，恐怕也是 c7w，Nauy，蒋委员长他们。

晚上回到宿舍后，我继续完成报告；似乎效率不如下午，倒是情有可原。最大的推进在于，我成功配好了 mac 上的 Latex 环境。由于四月份的某个清晨，我试图配置过，然而以失败告终，所以此次我听子睿说建议我本地配置，起初是畏难的；而后查看了一些中文博客，发现我几个月前就看过相关的资料；硬着头皮开始配环境。

其实配置相对简单，就是安装 MacTex 这个软件，实际上是四件套；而后在 VSCode 里安装插件。前者配好后需要调到 XeTex，而后者安装好插件后，需要改动 settings.json。这些完成后，还有些坑的地方，比如本地编译为了找到引用，需要反复编译数次，否则无法找到 ~/ref{} 语法指向的位置。另外，图片会莫名其妙无法对齐，我询问 Weikai Yang 和择邻后才学会了这些。解决方法非常粗浅，控制变量，先复制照片编译，而后一张张替换为所需的图片即可。

关于 VSCode 的配置，我只能说不够新手友好，毕竟配置 VSCode 实际上是个 json 存储的 dict，知乎上的教程只给出了从零开始后的配置结果，没有考虑到如果用户本身就有很多插件，那么该如何插入新的配置项；当然，也有可能作者认为插件多的都是程序员，不用讲的那么细致。

配置完成后，编译后的 pdf 有 20MB。没有用命令行编译，反而是用图形化界面编译的。然而，pdf 看了非常美观，心满意足和鲁大师，子睿还有阿飞讨论了起来。至于余下的训练的细节和代码解读，我明天上午列出 TODO，下午和思元讨论即可。

接着，我们讨论到具体的实验细节，比如我稍微能够理解的双层 Sigmoid 被 EuclideanLoss 橄榄的原因，便给他分享；此外还有双层没有用 residual 和 dropout 这件事情，他问我 residual 是什么，其实我不知道，但是碍于我的虚荣，我无耻地说“在 RNN 中有，你可以去看看”。他看到我设置的 momentum，笑道“你这不是什么都没用吗？”

接着，我们讨论了激活函数的影响，这也是我稍微熟悉的，而我想请教的是损失函数的影响。在损失函数上，思元的见解让我颇为叹服——

  1. Softmax 的图像近似与经过平滑处理后的 Hinge，故而能够推断二者的训练效果相当。
  2. Euclidean 的结果相比 Softmax 较差，这可以从训练目标角度进行理解。Euclidean 的训练目标是希望线性层输出的结果尽可能接近真实值，也即在一个 \(\Re^n\) 空间上的输出结果尽可能接近一 个 one-hot 向量；而 Softmax 的目标是线性层输出的结果映射到概率分布空间上尽可能接近真实值，也即在一个 \(\Re^n\) 空间上的输出结果映射到一个概率分布空间后尽可能接近一个 one-hot 向量，考虑到概率分布空间本就是一个各维度之和为 1 的 \(\Re^n\) 子空间，这与 one-hot 分布的空间是完全相同的。故而 Softmax 损失函数的训练目标更为优化，能够理解更好的训练效果。举例而言，如果线性层最后输出向量 \((0,0,0,100)^T\)，而期望的 one-hot 向量是 \((0,0,0,1)^T\)​ 。对于 Euclidean 而言，这个输出结果非常糟糕，损失值大，模型会惩罚当前的参数，使得输出结果的最后一项降低直至接近 \((0,0,0,1)^T\)；然而，对于 Softmax 而言，这个输出结果非常好，经过 Softmax 后，线性层的输出向量高度接近 \((0,0,0,1)^T\)，这一损失值很小，模型会倾向于奖励当前的参数。
  3. Euclidean 虽然训练目标和 one-hot 分类并不相近，最终会遏制模型的表达能力，然而 Euclidean 在训练初始时的损失值较大，收敛速率相对更高，这能够从 Euclidean 和 Softmax 的训练正确曲线梯度的对比中可以看出。

他说这不是他的 insight，而是他听了 CS231N 后的结果，都是 231N 讲过的。而他上个寒假听了 2 周左右就结束了，还做了作业。然而，我也试图听过 CS231N，然而讲到 CUDA CUDNN 和框架后，我就没法坚持下去了，甚是惭愧。

  • 晚上还有个破防事情，我和 uy 在 C 楼解决 wandb；突然发现我的 zsh 寄了，zsh: parse error near ‘)’；倒腾了特别久，心态崩了；Mac 是我生命的延续，看到他用不了了，我感到非常痛苦，一行一行 debug，我还叫来了鲁大师，害怕我中途心态蹦掉，从一楼跳下去；我们用二分 debug，然后还发现 source activate 寄了；报错说是某个 conda bash 脚本寄了，我试图做了一系列修改，完全没成功，彻底破防；然而，没有解决，我怀疑是编码问题，然而不是；一行一行 debug，最后发现跪在了 alias fi="find ./ -name '.DS_Store' -depth -exec rm {} \;" 上，估计 fi 是关键字，而我这个 alias 彻底干扰了系统；这让我意识到，mac 也有些糟粕的时候，windows 绝非一无是处
  • 晚上我，Nauy 还有鲁大师一起散步。Nauy 说的好，我的确能洞察人心，然而人性险恶，自己该多多注意所言所写，至少需要默契地留下来；我们谈到系里其他同学的工作，我还是那个观点，什么是虚名什么是实干，自己太清楚了，我没什么羡慕别人的，我只能做好我的工作；此外，对于 NLP 该有更多的了解。是非成败转头空，今朝汝笑尔，明朝又和妨呢？
