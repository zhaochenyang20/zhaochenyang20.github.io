GitKraken 使用文档

发表于2022-01-29|更新于2022-03-07|全栈开发
|字数总计:204|阅读时长:1分钟|阅读量:

初始化

  • 由于我用的是 mac ，可能略微有区别，但是问题应该不大。

  • 软件没有汉化版，凑合着用。

  • 点开软件后突然要求我重新生成 ssh key，给我吓了一跳。貌似绕不过这一层，于是决定重新生成了 key，然后确定了实验室的服务器还能用，并且在 github 上添加了自己新的 key。

  • 然后一同初始化，最后我选择了更图形化的界面。

使用

  • 受到了这个视频的启发才觉得试试 gitkraken
  • 个人感觉比较方便，github 和 gitlab 都能用。貌似之前用 github desktop，对 gitlab 支持不太行，而且就没有可视化，就帮用户节约了命令行
  • 克隆还是比较方便，可视化程度相当好
