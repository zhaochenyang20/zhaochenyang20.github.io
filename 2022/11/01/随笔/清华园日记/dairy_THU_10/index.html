但行好事，莫问前程

这么久以来，每当我的研究受阻，而后有了一些新的启发后，我便会想起这句话。无数次失意自责之后，突然意识到了自己的问题，欣喜若狂，我又在我的边界上有了突破。

昨晚（10.31），我在开会前半小时搓出了第一版的实验框架，而后交到了公司的主机上，结果开完会议，再和安仔聊了聊，回到宿舍，发现实验没有成功，而且 TeamViewer 的连接也断了。我感到非常诧异，心里却非常着急而焦虑，希望能够当晚把实验跑出来，而第二天可以写报告。

于是，我借来了室友的电脑，一台崭新的 3070，8G 的显存让我垂涎…开始跑实验，却几乎重写了整个批量测试的框架。然而，实验的结果并不顺利，其一是 Windows 的路径需要用到 “\\”，而这只能用 split("\\") 来完成；其二，Windows 上遇到了一些奇怪的问题，Python 给的报错是 os error；其三，也是最大的教训，我居然写出了这样的调试指令：

我一向嘲笑会在 Python 里写出死循环的行为，而昨晚这一串指令跑了 1min 后还没结束，我才意识到自己写了如此可笑的死循环。

原来，我心急的时候，什么错误都可能发生。

到了今天（11.01），我还在修改实验框架。早晨起来让朝晖哥重启了主机，发现果然的我的死循环导致了 Ubuntu 系统错误。继续改框架，尝试不同的 batch_size。

从事深度学习相关工作的研究者与工程师们都有着这样的癖好——他们渴求找出最精确的参数，让显卡被榨干到一丝不剩。

深度学习——她比政治学更实事求是，比数学更脚踏实地，比体系结构更理想主义，比软件工程更简洁优美。深度学习就是优雅，严谨，踏实，无私的代名词。

我接着尝试不同的 batch_size，发现越发反常，到了最后居然 model 已经无法 load 进入显存，让我感到非常绝望，怀疑人生。突然间，我似乎想起了一件事情——

若干多个月前，我还在 AIR 时，有一次跑实验，田哥劝我不要 ctrl + c 连按下很多次，因为第一次按下时，主进程就会被杀死，而后开始回收子进程的空间，多次按下 ctrl + c 可能会导致子进程无法回收，程序结束了，CUDA 还被占用。再次打开 nvidia-smi，果然如此。

杀灭了死进程后，果然能够 load 了。看来，大二学年在 AIR 的工作，的确教会了我一些事情。我经常怀疑大二一年自己碌碌无为，可能是见到了周围同学多少都有了工作的发表，而我似乎还看不到任何希望。我真的什么都没有学会吗？并没有，我一定是学会了什么，只是在同辈压力下感到非常焦虑。

最后，我尝试寻找高效杀死死进程的方法，却无功而返，反而因为用了 killall -p <username> 再次让 Ubuntu 崩溃了…第三次，我再次杀进程，这次杀了 TeamViewer，再次和主机断开了连接…

第四次拜托朝晖哥打开了主机后，我的心平静了下来——我发觉了一件事情：

你太心急了…

昨天没把实验跑出来会如何？今天没把报告写出来会如何？信原欠下的课还没学会如何？没法在暑研前做出自己满意的工作会如何？

天会塌下来吗？地球会停止运作吗？你的朋友会离开你吗？明天就没法活下去了吗？

并不是的，昨晚我如此给 Anki 讲，劝她不要焦虑，踏实做一些事情，培养研究的品味并且夯实以后做研究的基础；转眼想了想，我自己似乎也没有做好这些事情，我在为了很多事情而焦虑，我四处劝告我的朋友们，不要焦虑，我们都有光明的未来，转而发现，我才是最焦虑的那一个，我潜在的意识里总有危机感，这样的危机感已经演化成了深深的焦虑。

你太焦虑了，这不是做好工作的状态，我当平复下来，收拾自己的内心，潜心钻研。

但行好事，莫问前程

碎语

十月开始于成外的聚餐，与新入学的成外学弟学妹们聚在一起，我猛然意识到了自己又成了校园里的所谓高年级，我理应很成熟，很稳重，对未来有把握，对过往有思索。

也许如此，却又不完全如此。大二一年我保持着极其高的学习率，极其痛苦地在政治、学业、研究和人际关系上随机梯度下降，我以为到了大三，就几近收敛，其实远没有——我不知这是好事，还是新一年痛苦挣扎的预兆。

在政治上，漫长的十月，发生了数个的政治事件，无外乎是关于疫情和二十大，却没有一件看上去是向着好的方向发展，我也大量阅读了一些民主相关的书籍与论著——似乎民主并没有那般的伪善，而我一向认为，中国的情况只适合接受集权化的共产主义（我不能在这样的体系下看到一丝民主的身影，哪怕精英民主也不是），导向资本主义只会沦为外部市场而再次遭受百年的剥削。然而，我必然是有什么重要的事情并未看见——可能出于我对那些只知民主表皮不知唯物主义的天龙人的唾弃，也可能是我主动在很长一段时间避开了任何民主思潮与观点——总之，我试图在政治上见识更多的可能，否则大陆反智的疫情管控只会让我陷入绝望。

学业上，我试图去抢救一些课程未果，然而我的确做完了通识课的工作，能够为期末省下很多心力，这是很好的事情。至于研究，这是最重要的——商汤的工作真的是目前为止我对职业最美好的想象，业界，研究，创造，氛围与生产力。我甚至一度产生了“如果能留在商汤工作，本科毕业就业也很好。”可惜，这样的事情答题不会发生，不过我非常喜爱这样的研究机会，也让我觉得十分幸福，距离 Ze 所言的在研究上的开悟，我想来是越来越接近。

人际关系上，我似乎已经进入了平稳的悲观阶段，这不一定是坏事，不过可见长度的时间内，我将会保持着这样的平稳。世界太乱，未来太远，我会去向何方全然不知，对羁绊带来的约束的恐惧远超出了我对羁绊本身的渴求。

絮絮叨叨，说了这么多 Nonesense，就这样开始 11 月的生活…

Sihan Wang 的想法

但是话题再说回来，在现在的大环境下，我们应该如何表态，怎么发声?
我想了很多，最后结合王志安总结天安门广场上的惨剧的感悟以及今晩老师的这一系列话，大概有以下的几个准则和注意事项：

  1. 保护好自己的人身安全，保护好一个给彼此进退的灰色地带。

  2. 请理性发声，以逻辑和法理制胜，动员身边人的力量。我认为学生运动存在的最大问题其实是大家更多地被情渚所调动，而不是理性地去判断这件事本身的对错。诚然，根据《乌合之众》这本书的说法，群体的判断力是要比个人弱很多的，所以要求一整个群体去理性是十分困难的。但在我们作为有责任感的青年，请好好地掂量发声权力以及其所义务，做理性的推演，做逻辑的奴隶。

  3. 保留所有人辩论的机会。亚里士多德所言”吾爱吾师，吾更爱真理“可供大家参考。无论政治立场是否统一，信息差多大，请保留别人跟自己辩论，从而有了达成共识的机会。请在辩论的时候尊重对手，就算他秉持的是你厌恶的言论和思想。发声的权力和执法的权力又有什么不同呢，更自由的言论就意味着更高的道德要求。

  4. 请多读书，多看历史，谦逊有礼。Manners Make the Man!

最后，虽然最近政治抑郁的厉害，但我仍对这个国家这个民族抱有最深沉的热爱。中国的言论自由建设道路任重而道远，还是请诸君努力奋斗，用正确理性的方式建设一个更好的末来!

道路曲折我走不完，前途光明，我们会看见吗?

研究品味

这几天一字班小朋友开始问我进组的事情，凡是不能说的，我向来缄默不言，问就是我不知道。

这些话是我博客右下角的赛博猫猫生成的，和我没关系，如有雷同，纯属雷同….

我比较推荐的符合我研究品味的老师：

  1. 段岳圻——电子系的 New AP，年轻的 CVer，乐于带本科生，研究品味不错，做出的工作算是十年磨一剑的大作；我在 AIR 还写过人家的报告，可惜当时理解 CV，以及有眼不识泰山…
  2. 崔鹏——数据挖掘方向的好老师，去过的都说好，可惜太小众了，研究品味也不错
  3. 王运涛——HCI 的小老板，人很好，工作 Profound，和 UW 有千丝万缕的联系，这个就不多说了，懂得能懂…
  4. 朱军——朱老师是贵系之光，研究的是机器学习，不是我这种炼丹的人能高攀的；具体而言，是通过机器学习来解决数学物理问题。整个组非常强，毕竟是今年特奖松铭哥的组。美中不足，只招强者，能力较弱不太适合
  5. 孙茂松——这才是我大二应该的去处，哎，都是血泪史…孙老师组氛围是很好的，而且乐于带人，八字班特奖是他的关门弟子了。美中不足，孙老师快退休了，不招学生了，把 THUNLP 传给了刘知远
  6. 龙明盛——软院的神，清华之光，对数学和深度学习都有很深的 insight，也比较带人；美中不足，不太收人，无论是外系学生还是低年级

高性能方向我不了解，不过去的人挺强的，都是数理基础扎实或者学过 OI 的。还有我室友宇晨，非常聪明…
